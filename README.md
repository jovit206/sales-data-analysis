# Sales Data Analysis Project

## Tools Used
- Python (Pandas, Matplotlib)
- MySQL
- Machine Learning (Scikit-learn)

## Description
Analyzed sales data to find business insights, stored data in MySQL, and built a machine learning model to predict profit.

## Features
- Data cleaning and analysis
- Visualization of sales trends
- SQL database integration
- Profit prediction using regression

## Outcome
This project demonstrates end-to-end data analysis using Python, SQL, and ML.
